PROJETOS

PROJETO #01

Pagina de Login

A página de login foi desenvolvida para permitir que usuários acessem o sistema de forma rápida, segura e intuitiva. Seu layout é simples e objetivo, priorizando a facilidade de uso e a clareza das informações.O formulário contém campos essenciais para autenticação, como usuário/e-mail e senha, além de botões de ação bem destacados. PROJETO #02

Site Cardápio

O projeto consiste no desenvolvimento de um site de cardápio digital responsivo, criado para permitir que restaurantes, lanchonetes e estabelecimentos gastronômicos apresentem seus produtos de forma organizada, atrativa e acessível em qualquer dispositivo.O site foi projetado com foco em usabilidade, performance e experiência do usuário, garantindo navegação simples, carregamento rápido e visual moderno. O layout prioriza a hierarquia visual das informações, destacando categorias, preços, descrições e imagens dos produtos.Além disso, o sistema possibilita futuras expansões, como integração com pedidos online, WhatsApp, QR Code e painel administrativo para atualização de itens.
